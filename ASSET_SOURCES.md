# 第三方素材来源

本项目下载并归档了以下 Kenney 素材包，均标注为 Creative Commons Zero（CC0）：

| 素材包 | 用途 | 来源 |
|---|---|---|
| Pixel Car Pack | 车辆与行人 | https://kenney.nl/assets/pixel-vehicle-pack |
| Pixel UI Pack | 像素 UI 备选素材 | https://kenney.nl/assets/pixel-ui-pack |
| Background Elements | 背景元素 | https://kenney.nl/assets/background-elements |
| Platformer Pack Industrial | 工业场景与施工道具 | https://kenney.nl/assets/platformer-pack-industrial |
| Particle Pack | 火花、碎屑和环境粒子 | https://kenney.nl/assets/particle-pack |
| Interface Sounds | 点击、确认、错误提示 | https://kenney.nl/assets/interface-sounds |
| Impact Sounds | 木材和金属撞击/断裂 | https://kenney.nl/assets/impact-sounds |

每个解压目录均保留原始 `License.txt`。正式游戏当前直接使用车辆包中的 `sedan.png`，以及 Interface Sounds、Impact Sounds 中选取的 `.ogg` 文件。
